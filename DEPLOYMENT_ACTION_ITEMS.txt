❌ TROUBLESHOOTING

If frontend shows 404:
  1. Check Vercel deployment status (green checkmark?)
  2. Check build logs for errors
  3. Make sure environment variables are set

If frontend connects but shows errors:
  1. Open browser F12 → Console tab
  2. Look for error messages
  3. Check Network tab for failed API calls
  4. Verify backend domain is correct

If backend returns 500 error:
  1. Check backend environment variables in Vercel
  2. Verify DATABASE_URL and JWT_SECRET are set
  3. Check backend deployment logs
